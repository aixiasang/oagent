🤖 OAgent智能体框架
====================

🧩 核心功能
----------
- **工具调用系统**：支持 20+ 系统/网络/文件操作工具
- **流式响应**：实时显示 AI 思考过程
- **Web 界面**：现代化聊天界面
- **意图分类**：智能路由用户请


💡 终端模式
---------
运行 `python agent/programor.py` 直接在终端与 AI 交互

🔄 交互流程
--------
1. 用户发送消息
2. 系统进行意图分类
3. 判断是否需要工具调用
   ├── 是 → 执行工具 → 返回结果
   └── 否 → LLM 直接生成回复
4. 流式输出结果到界面

🔧 扩展能力
---------
- 在 `tools/` 添加新工具实现功能扩展
- 在 `prompts/` 修改提示词优化回答质量
- 在 `agent/` 开发新代理类型
